## 4. Rust (The Future Systems Language)

```
What's happening:
  Rust is replacing C/C++ everywhere. Memory safety without garbage collection.
  Used by: AWS (Firecracker), Cloudflare, Discord, Meta, Microsoft, Linux kernel.

Why learn Rust:
  - Memory safe (no segfaults, no data races) at compile time
  - As fast as C/C++
  - Growing ecosystem (web, CLI, embedded, blockchain)
  - WebAssembly target (run in browser at near-native speed)
  - High demand, low supply → premium salaries

Use cases in 2025:
  - Cloud infrastructure (Kubernetes operators)
  - Databases (TiKV, SurrealDB)
  - CLI tools (ripgrep, bat, exa)
  - WebAssembly modules
  - Blockchain/crypto (Solana is Rust)
  - Game engines

Learn it if: You want to work on infrastructure, databases, or performance-critical systems
```
